# mynotes

## Project setup
```
install docker desktop
```
### Change IP-Address
```
open ./frontend/src/store/index.js
Change the localhost Variable in State to your machines IP-Address + :3000
```

### Run App
```
cd frontend
docker-compose up --build
```

### Open App
````
enter Your IP_Address + :3000 in your Browser.
````

### Done
